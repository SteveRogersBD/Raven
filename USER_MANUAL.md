# PlateIt User Manual

Welcome to **PlateIt**, your AI-powered kitchen companion! This guide will help you get started with the app, from installation to cooking your first meal.

### 1. Installation
*   **Download:** Get the latest `PlateIt.apk` from the [Releases](https://github.com/SteveRogersBD/PlateIt/releases) page (or the link provided).
*   **Install:** Open the file on your Android device. If prompted, enable "Install from Unknown Sources" in your browser or file manager settings.
*   **Launch:** Open the app from your home screen.

### 2. Account Setup
*   **Sign Up:** Create a new account using your email and a password.
*   **Preferences:** Select your favorite cuisines (e.g., Italian, Mexican, Asian) to help PlateIt personalize recommendations.
*   **Sign In:** If you already have an account, simply log in.
